Class: ScaleManager

ScaleManager

ScaleManager

new ScaleManager(width, height)

The ScaleManager object is responsible for helping you manage the scaling,
resizing and alignment of your game within the browser.
Parameters:
Name Type Description
width number The native width of the game.
height number The native height of the game. v1.0 Initial version
